# Web — Telegram Sender Pro

Рабочая панель управления рассылкой.

## Структура

```
web/
├── index.html      # главная страница
├── app.js          # логика + вызовы API
├── assets/
│   └── app.css     # стили
└── README.md
```

Демо kimi.page (без API) перенесено в `archive/kimi-demo/`.

## Запуск

Из корня проекта:

```bash
./run-web.sh
```

Браузер: **http://127.0.0.1:8000**

API и фронт на одном порту — `app.js` ходит на `/api/...`.

## Разделы

- **Дашборд** — статистика
- **Аккаунты** — вход по коду, проверка
- **Получатели** — список для рассылки
- **Новая рассылка** — запуск кампании
- **Рассылки** — лог и прогресс
